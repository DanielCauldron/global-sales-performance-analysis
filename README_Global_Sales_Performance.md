# 🌍 Global Sales Performance Analysis -- Power BI

## 📌 Contexto de Negócio

Este projeto foi desenvolvido com foco em análise executiva de vendas
globais, simulando o ambiente de uma empresa multinacional.

O objetivo é fornecer visão estratégica sobre: - Crescimento anual (Year
over Year - YoY) - Performance por mercado - Rentabilidade por
categoria - Sazonalidade de vendas

------------------------------------------------------------------------

## 📊 Indicadores Desenvolvidos

-   💰 Faturamento Total
-   📈 Lucro Total
-   📦 Total de Pedidos
-   📊 Faturamento Ano Anterior
-   📉 Crescimento Anual (%)
-   📆 Análise temporal com Tabela Calendário

------------------------------------------------------------------------

## 🧠 Principais Insights

-   Crescimento anual de 22%.
-   Mercado APAC lidera em volume de vendas.
-   Forte sazonalidade no último trimestre.
-   Categoria Technology apresenta maior potencial estratégico.

------------------------------------------------------------------------

## 🛠 Tecnologias Utilizadas

-   Power BI
-   DAX
-   Modelagem Dimensional (Star Schema)
-   SAMEPERIODLASTYEAR
-   Cálculo de Crescimento YoY
-   Power Query

------------------------------------------------------------------------

## 🎯 Diferenciais Técnicos

-   Implementação de Tabela Calendário
-   Comparação Ano a Ano com inteligência temporal
-   KPI executivo com análise percentual
-   Estrutura orientada à tomada de decisão
